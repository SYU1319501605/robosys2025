## chrcountコマンド ##


## 概要 ##
標準入力から読み込んだ文字列の中に含まれる以下の数をカウントします：

- **英字（A–Z, a–z）**
- **数字（0–9）**
- **その他の文字**

---

## 使い方 ##

### 基本 ###

```bash
$ echo  'Hello123!' | ./chrcount

total     9
alphabets 5
digits    3
others    1
```

## 仕様 ##

- 入力は標準入力から受け取る
- 結果を標準出力に表示する
- パイプ処理に対応
- 空入力時はエラーメッセージを表示
- 日本語などマルチバイト文字も「others」としてカウント
